# 仕入れ・販売差異分析ダッシュボード

仕入れ数量と販売数量のデータを分析し、在庫差異、品切れリスク、売上効率などを可視化するWebダッシュボードアプリです。

## 📦 ファイル構成

```
.
├── dashboard_app.py              # Streamlitアプリ
├── 野菜_仕入れ数量表.xlsx         # サンプル仕入れデータ
├── 野菜_販売数量表.xlsx           # サンプル販売データ
└── README.md                     # このファイル
```

## 🚀 インストール & 実行方法

### 1. 必要なライブラリをインストール

```bash
pip install streamlit pandas plotly openpyxl
```

### 2. ダッシュボードを起動

```bash
streamlit run dashboard_app.py
```

### 3. ブラウザで自動的に開きます

ブラウザで以下のURLが自動的に開きます：
```
http://localhost:8501
```

## 📊 機能一覧

### 1️⃣ **KPI サマリー**
- 総仕入れ数量・金額
- 総販売数量・金額
- 現在の在庫量
- 利益・利益率
- 未販売品目数

### 2️⃣ **グラフ分析**

#### 📈 仕入れ vs 販売タブ
- **品目別比較**: 仕入れ数量と販売数量の棒グラフ
- **日付別推移**: 日時経過による仕入れ・販売の折れ線グラフ

#### 📊 品目別分析タブ
- **ステータス分布**: 円グラフで差異の分布を表示
- **品目別在庫量**: 在庫過剰（赤）/ちょうど（緑）を色分け

#### 💰 金額分析タブ
- **仕入れ金額 vs 販売金額**: 品目別の金額比較
- **売上効率**: 販売数量 / 仕入れ数量 の比率

#### 📦 在庫状況タブ
- **在庫推移**: 日付による在庫量の変化を折れ線グラフで表示

### 3️⃣ **詳細テーブル**
- 全データを一覧表示
- **ステータス別フィルター**: 在庫過剰 / 品切れ / 未販売 / ちょうど
- **品名でフィルター**: 特定の野菜に絞り込み
- **ソート機能**: 日付、品名、数量、在庫など自由にソート

### 4️⃣ **エクスポート機能**
- **CSV ダウンロード**: 分析結果をCSVで出力
- **Excel ダウンロード**: 分析結果をExcelで出力

## 🔍 分析指標の説明

| 指標 | 説明 |
|------|------|
| **仕入れ数量** | 商品の仕入れ量（kg） |
| **販売数量** | 実際の販売量（kg） |
| **在庫量** | 仕入れ数量 - 販売数量 |
| **ステータス** | 在庫過剰 / 品切れ / 未販売 / ちょうど |
| **仕入れ金額** | 仕入れ数量 × 仕入れ単価 |
| **販売金額** | 販売数量 × 販売単価 |
| **利益** | 販売金額 - 仕入れ金額 |
| **利益率** | (利益 / 販売金額) × 100 |
| **売上効率** | (販売数量 / 仕入れ数量) × 100 |

## 🎯 ステータスの分類

```
在庫過剰  → 仕入れ > 販売（売れ残りがある）
品切れ    → 販売 > 仕入れ（売上を逃している）
未販売    → 販売数量 = 0（全く売れていない）
ちょうど  → 仕入れ = 販売（完売）
```

## 💡 使用例

### 例1: 在庫過剰な商品を確認
1. 左側の「ステータスでフィルター」で「在庫過剰」を選択
2. 品目別在庫量グラフで赤い棒を確認
3. 詳細テーブルで数量と金額を確認

### 例2: 売上効率が低い商品を特定
1. 「金額分析」タブで「売上効率」を確認
2. 効率が低い品目の仕入れ量を削減検討

### 例3: 期間別の分析
1. 詳細テーブルで「日付」でソート
2. 特定の期間のデータをフィルター
3. CSVやExcelでダウンロード

## 🐛 トラブルシューティング

### エラー: "Module not found"
→ 必要なライブラリをインストールしてください
```bash
pip install streamlit pandas plotly openpyxl
```

### グラフが表示されない
→ ブラウザのキャッシュをクリアしてから再度アクセス

### 大量データで遅い
→ データを日付範囲でフィルターして使用量を減らしてください
